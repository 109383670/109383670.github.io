v-on

v-html

可以将双括号内的值输出为html格式，不然会一直是字符。
你的站点上动态渲染的任意 HTML 可能会非常危险，因为它很容易导致 XSS 攻击。请只对可信内容使用 HTML 插值，绝不要对用户提供的内容使用插值。

快速原型开发

vue serve

在.vue文件的目录下运行这个命令，可以建立一个简单的本地服务器。会自动生成一个node_moudle文件夹。
它会在当前目录自动推导入口文件——入口可以是 main.js、index.js、App.vue 或 app.vue 中的一个。你也可以显式地指定入口文件：

vue create

创建一个项目。
创建完项目后，可以在项目文件夹进行调试。

vue ui

打开一个 Ui工程配置管理器。

vue build
